## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/indapa/SolarProduction.git
cd SolarProduction
```

2. Install uv

```curl -Ls https://astral.sh/uv/install.sh | bash```

3. Recreate environment:

```
uv venv
uv sync
```
4. Run the app:

```
uv run streamlit run Solar_Production.py
```
